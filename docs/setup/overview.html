NOTE

This is an independent open-source project and is not affiliated with or endorsed by Google. The project name and features are intended to integrate with Home Assistant/Lovelace and do not represent an official Google product.

Material Home Component for Home Assistant is a suite of custom UI elements designed to bring the look and feel of the Google Home app directly into your Home Assistant dashboard.

The project implements the principles of Material Expressive 3, the latest evolution of Google's design system. This ensures a modern, fluid, and smart home-optimized interface characterized by dynamic shapes and a clear visual hierarchy.

Unlike traditional themes, it is more than just a visual layer: it is a fully modular system of interactive cards, custom icons, and dynamic layouts. Every component is built specifically for Home Assistant with a rigorous focus on responsiveness, touch-first navigation, and entity-driven customization.

⚠️ Important

This is not a theme, but a collection of custom cards that integrate with Material You themes, adding buttons, sliders, dashboards, and dynamic controls to your UI.

🧠 Purpose ​

The main goal is to provide a modern, touch-friendly, and modular interface for Home Assistant that is clean, elegant, and intuitive on both desktop and mobile devices. Whether you are managing lights, climate, cameras, or network devices, this suite offers a fluid and visually consistent experience across your entire smart home.

✨ Key Features ​

📱

Mobile-First UX

Optimized for phones and tablets with fluid touch interaction and adaptive sizing.
🎨

Material Expressive 3

Full support for light/dark modes, dynamic color palettes, and modern rounded shapes.
🧩

Custom Modular Cards

Ready-to-use components for lights, climate, Wi-Fi, cameras, and more.
⚙️

Smart Configuration

YAML-based setup with intelligent defaults, fully compatible with UI editors.
  • 🔁 Dynamic Behavior Entity-aware styling, conditional rendering, and real-time updates for a living dashboard.
  • 📦 Flexible Installation Choose your preferred method: via CDN or manual installation from GitHub.
  • 🔄 Seamless Updates Full HACS compatibility for effortless, one-click updates.
