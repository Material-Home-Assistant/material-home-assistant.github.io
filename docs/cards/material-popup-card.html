Material Popup Card 📱 ​

The material-popup-card is a highly customizable popup card for Home Assistant Lovelace dashboards. It is designed to provide a native-like experience with smooth animations, hash-based navigation, and extensive styling options. ✨

Features 🌟 ​

  • Hash Navigation: Open popups using URL hashes (e.g., #popup-light), allowing for easy linking and back-button support. 🔗
  • Native Feel: Smooth slide-up animations and "slide-to-close" gestures on mobile. 👆
  • Customizable Header: Optional header with title, icon, toggle button, and sub-buttons. 🎨
  • Sub-buttons: Add quick actions directly in the popup header. 🔘
  • Styling Options: Customize width, background color, blur effects, and border radius. 💅
  • Visual Editor: Fully configurable via the Lovelace visual editor. 👁️

Configuration 🛠️ ​

Here are the configuration options for the material-popup-card:

NameTypeDescriptionDefault
typestringcustom:material-popup-cardRequired
hashstringThe URL hash that triggers this popup to open. (e.g. #popup) 🔗#popup
cardslistList of cards to display inside the popup. 🃏[]
headerobjectConfiguration for the popup header. See Header Options.
settingsobjectGeneral settings for the popup behavior. See Settings.
styleobjectStyling options. See Style Options.

Header Options 🎩 ​

NameTypeDescriptionDefault
show_headerbooleanWhether to show the header. 👁️true
namestringTitle displayed in the header. 🏷️
iconstringIcon displayed in the header. 🖼️
button_typestringType of the main header element: switch, state, or name.name
entitystringEntity to control if button_type is switch or state. 🆔
sub_buttonslistList of sub-buttons to display in the header. See Sub-buttons.[]

Settings Options ⚙️ ​

NameTypeDescriptionDefault
auto_closenumberTime in milliseconds to automatically close the popup. ⏱️
close_by_clicking_outsidebooleanClose the popup when clicking the backdrop. 🌑true
close_on_clickbooleanClose the popup after clicking anywhere inside it. 🖱️false
slide_to_close_distancenumberDistance in pixels to drag down to close the popup (mobile only). 📏150

Style Options 🎨 ​

NameTypeDescriptionDefault
width_desktopstringWidth of the popup on desktop screens. 🖥️540px
width_mobilestringWidth of the popup on mobile screens (if custom mode). 📱100%
bg_colorstringBackground color of the popup. 🌈
backdrop_blurnumberBlur amount for the backdrop (in px). 🌫️10
border_radiusstringBorder radius of the top corners. 📐42px

Sub-buttons 🔘 ​

You can add small buttons to the header for quick actions.

yaml
header:
  sub_buttons:
    - icon: mdi:power
      tap_action:
        action: toggle
      entity: light.kitchen

Examples 📜 ​

Basic Popup ​

yaml
type: custom:material-popup-card
hash: #my-popup
header:
  show_header: true
  name: My Popup
  icon: mdi:lightbulb
cards:
  - type: entities
    entities:
      - light.living_room

To open this popup, you can use a button card or any action that navigates to #my-popup.

yaml
type: button
name: Open Popup
tap_action:
  action: navigate
  navigation_path: "#my-popup"

Advanced Usage 🚀 ​

Browser Back Button 🔙 ​

The popup integrates with the browser's history. Pressing the "Back" button on your device or browser will close the popup instead of navigating to the previous page.

Mobile Slide-to-Close 👆 ​

On mobile devices, you can drag the popup header downwards to close it. This gesture is disabled if you start dragging from the content area to prevent conflicts with scrolling.
